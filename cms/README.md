# Warga CMS

**Command Center** untuk administrator (RT/RW/estate): manajemen warga, IPL, berita, UMKM, dan lokasi.

- **Stack:** React 19, Vite, TypeScript, Tailwind CSS v4, **[shadcn/ui](https://ui.shadcn.com/)** (komponen + token desain).
- **Perencanaan produk:** lihat [`docs/CMS-PLANNING.md`](../docs/CMS-PLANNING.md) di root monorepo, atau halaman **Perencanaan CMS** di sidebar aplikasi ini.

## Perintah

```bash
npm install
npm run dev    # http://localhost:5173
npm run build
npm run lint
```

Jalankan bersamaan dengan app warga (`../web`); gunakan port berbeda bila perlu (`npm run dev -- --port 5174`).
